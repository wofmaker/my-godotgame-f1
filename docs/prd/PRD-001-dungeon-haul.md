# PRD: Dungeon Haul — 던전 루터 서바이버

## 목적
타르코프의 루팅/로스트 긴장감과 뱀서의 자동전투 중독성을 결합한 2D 탑다운 로그라이트. "한 층만 더..."의 유혹과 "지금 빠질까"의 긴장감이 핵심 경험이다.

## 사용자 스토리
- As a 로그라이트 팬, I want 매 런마다 다른 던전 구조와 스킬 조합을 경험하고 싶다, so that 반복 플레이에도 신선함을 느낄 수 있다
- As a 서바이버 팬, I want 이동에 집중하면서 자동으로 적을 처치하고 싶다, so that 직관적 조작으로 전투의 쾌감을 느낄 수 있다
- As a 루터 팬, I want 위험을 감수하고 더 깊이 내려가 희귀 아이템을 얻고 싶다, so that 리스크/리워드 의사결정의 스릴을 느낄 수 있다
- As a 기지 건설 팬, I want 던전에서 얻은 자원으로 기지를 확장하고 싶다, so that 런 사이에도 성장을 실감할 수 있다

## 기능 요구사항

### 핵심 기능
- [ ] F-01: 캐릭터 이동 — WASD 8방향 이동 + Space 대시(쿨다운 2초, 무적 0.3초)
- [ ] F-02: 자동 전투 — 범위 내 최근접 적에게 무기별 패턴으로 자동 공격
- [ ] F-03: 무기 시스템 — 검(부채꼴), 활(조준 투사체), 지팡이(관통). 기지에서 제작/선택
- [ ] F-04: 패시브 특성 — 5종, 슬롯 2개, 기지에서 해금/장착, 런 중 변경 불가
- [ ] F-05: 스킬 시스템 — 20종, 레벨업 시 3택, 최대 레벨 5, 런 종료 시 초기화
- [ ] F-06: 절차적 던전 생성 — BSP 기반 방 배치 + 방 내부 오브젝트 배치, 4층 구성
- [ ] F-07: 방 단위 전투 — 방 입장 시 적 일괄 스폰 → 전멸 시 문 개방, 추가 스폰 없음
- [ ] F-08: 적 AI — 근접형(추적), 원거리형(거리유지+투사체), 엘리트(특수패턴), 보스(층당 1종, 패턴 3개)
- [ ] F-09: 알람 게이지 — 상자+15%, 엘리트+10%, 방클리어+5%, 보스+25%. 90% 초과 시 무적 추적자 스폰
- [ ] F-10: 인벤토리 — 12칸(최대 20칸), 드래그앤드롭, 칸 부족 시 취사선택 강제
- [ ] F-11: 루팅 — 아이템 등급(일반/마법/희귀/유물/설계도), 드롭 테이블, 층별 가중치
- [ ] F-12: 탈출 시스템 — 1~2층 즉시, 3~4층 5초 채널링(피격 시 중단), 매 층 탈출 포탈 존재
- [ ] F-13: 사망 페널티 — 런 획득 전리품 전부 로스트, 기지/해금은 유지
- [ ] F-14: 기지 건설 — 확장형(소재로 영역 확장), 자유 배치, 인접 보너스, 건물 6종(대장간/마법연구소/창고/상점/회수반/여관)
- [ ] F-15: 이중 화폐 — 골드(범용) + 심연석(3~4층 희귀), 장비 제작/건물 건설에 사용
- [ ] F-16: 레벨업 — 적 처치 경험치 → 레벨업 시 일시정지 → 스킬 3택 → 선택 후 재개
- [ ] F-17: 세이브/로드 — 기지 상태, 해금 정보, 화폐 영속 저장 (JSON)

### 부가 기능
- [ ] F-18: 일일 도전 — 고정 시드 던전, 스코어 기록, 로컬 리더보드
- [ ] F-19: 미니맵 — 탐색된 방 표시, 포탈/보스방 아이콘
- [ ] F-20: 전투 이펙트 — 데미지 팝업, 히트스톱, 화면흔들림, 사망 슬로우모션
- [ ] F-21: BGM/SFX — 기지/던전 BGM 구분, 전투/UI 효과음

## 비기능 요구사항
- 아트: 프로토타입은 간단한 도형(원/사각형/삼각형)으로 구현, 최종 아트는 나중에 결정
- 성능: 방 내 적 15마리 + 투사체 동시 처리 시 60fps 유지
- 플랫폼: PC(Windows) 우선, 키보드+마우스 조작
- 저장: 자동 저장(기지 귀환 시), 슬롯 1개

## 구현 결정사항

| 영역 | 결정 | 근거 |
|------|------|------|
| 캐릭터 | 클래스 없음, 무기+패시브 특성 | 밸런싱 단순화, 빌드 다양성은 무기/특성 조합으로 확보 |
| 던전 생성 | 완전 절차적 (방 내부까지) | 무한 변형으로 리플레이 가치 극대화 |
| 적 스폰 | 방 입장 시 고정 수 일괄 스폰 | 방 단위 던전 구조에 적합, 무한 스폰 불필요 |
| 탈출 | 층별 차등 (즉시→채널링) | 깊을수록 탈출도 어려워지는 이중 리스크 |
| 기지 | 확장형 자유 배치 + 인접 보너스 | 시각적 성장 보상 + 전략적 배치 재미 |
| 경제 | 이중 화폐 (골드+심연석) | 깊은 층 도전 동기 강화, 인플레 관리 |
| 아트 | 도형 프로토타입 → 추후 교체 | 개발 속도 우선, 코어루프 검증 먼저 |
| 엔진 | Godot 4.x + GDScript | 2D 로그라이트에 최적, 무료 오픈소스 |

## 제외 범위
- 멀티플레이어/PvP — PvE 전용 확정
- 5층 + 탈출 런 — MVP 이후
- 히든 6층 / 무한 모드 — MVP 이후
- 단검, 도끼 무기 — MVP 이후
- 정찰탑, 훈련장 건물 — MVP 이후
- 온라인 리더보드 — 로컬만
- 모바일/콘솔 — PC 전용
- 다국어 — 한국어 단일
- 최종 아트 에셋 — 도형으로 진행

## 성공 기준
- [ ] 코어 루프 1회 완주 가능: 기지→던전 입장→4층 클리어→탈출→기지 귀환→장비 제작→재입장
- [ ] 사망/탈출 의사결정이 의미 있음: 3층 이상에서 "더 갈까 말까" 고민이 발생
- [ ] 기지 성장 체감: 10런 후 기지가 눈에 띄게 확장됨
- [ ] 빌드 다양성: 무기 3종 × 특성 5종 조합으로 최소 5가지 유효 빌드 존재
